# Ring Rush

一键换轨的手机小游戏：吃黄钻、避开红色闸门、冲高分。

Play: https://nono125-lee.github.io/ring-rush-game/

## 当前系统

- 难度已调整：比最初版本温和，但红色闸门与速度压力已上调到中等偏上。
- 体力系统：每天 3 次，用完后可通过“看广告+1”增加一次，不能提前囤体力。
- 排行榜系统：本机保存前 10 名成绩。
- 设置系统：可开关声音反馈和减少闪光效果。
- 结算提示：每局结束后按成绩显示鼓励或轻微嘲讽文案。

说明：网页版的广告按钮是演示逻辑；微信小游戏真广告需要在微信后台配置激励视频广告位 ID。
