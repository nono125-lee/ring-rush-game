# Ring Rush 微信小游戏版

这个目录是可导入微信开发者工具的小游戏工程。

## 本地预览

1. 安装并打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择本目录：`wechat-minigame`。
4. 没有正式 AppID 时可先用测试号/游客模式预览；正式上传前，把 `project.config.json` 里的 `appid` 改成你的小游戏 AppID。

## 正式发布前

- 在微信公众平台注册“小程序/小游戏”账号并拿到 AppID。
- 完成主体信息、类目、备案、隐私/用户协议等平台要求。
- 在微信开发者工具里上传代码。
- 到微信公众平台提交审核，审核通过后手动发布。

## 代码说明

- `game.js`：小游戏主逻辑，已从网页 DOM 版改成微信小游戏 Canvas 版。
- `game.json`：小游戏运行配置。
- `project.config.json`：开发者工具项目配置。
